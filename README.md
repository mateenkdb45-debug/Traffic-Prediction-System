# Traffic Prediction System

## Overview
The Traffic Prediction System is a web application developed using Python and Django. It helps predict traffic conditions and assists in scheduling content resources based on predicted traffic data.

## Features
- Traffic prediction
- User login and management
- Admin dashboard
- Data management
- Interactive web interface

## Technologies Used
- Python
- Django
- HTML
- CSS
- JavaScript
- SQLite

## Installation
1. Clone the repository.
2. Install the required packages:
   pip install -r requirements.txt
3. Run the server:
   python manage.py runserver

## Project Structure
- admins/
- users/
- templates/
- static/
- Utility/
- traffic_prediction/
